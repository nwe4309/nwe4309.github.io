GAME NAME

Development (August 2014 - September 2019)

Passion Project

Description:

• Random Cool things (RCT) is the mod that I developed for the game Terraria.
• The project was primarily worked on off and on, as well as being my first real introduction into learning to program.
• The mod is fully functional and was done using an open source API called tModLoader and programmed in C#.
• The mod added a few dozen unique weapons, tools, blocks, and crafting materials to the game.
• Each item has a unique function that sets it apart from the in-game items and designed to enhance and diversify the players experience without distrupting the base gameplay.
• I was the sole programmer creating all the weapons and weapon patterns. Art was a collaboration with a game artist for the in-game sprites.

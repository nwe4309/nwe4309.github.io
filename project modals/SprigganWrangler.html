Spriggan Wrangler

Development (October - November 2019)

School Project

Description:

• Spriggan Wrangler is a casual puzzle game set in the world of Ethshar.
• Spriggan Wrangler was a group project consisting of myself and four others.
• This game was designed and developed across the span of five weeks for one of the three games required for the class Game Design & Development 2.
• The game was developed in Unity using C# and I was one of two other programmers.
• My specific contributions to the project was the creating of level 2 and level 5, the programming of the levels being read in from a text file and then
placed in the scene. The programming of the level select screen and save feature that goes along with it, the dynamic spell bar that scales to the number
of different spells specified in the text file, the programming of tiles having a highlighted look when hovering over them, and finally the creation of the
fast forward mechanic which can be triggered any time by pressing the space bar or the fast forward button in the bottom right of the game.

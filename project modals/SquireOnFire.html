Squire On Fire

Development (January - April 2018)

School Project

Description:

• Squire on Fire was a group project between two other people and I.
• The game was developed in Monogame using Visual Studios and C#
• This was a class project in which we were tasked in creating a game in 10 weeks.
• Squire on Fire is a medieval themed bullet hell in which the player must defeat a single boss.
• My role was lead programmer and I handled creating all the classes, the general structure of the game loop, and the creation of all boss attack patterns.
